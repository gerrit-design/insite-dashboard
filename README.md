# Insite Dashboard

A comprehensive financial intelligence dashboard for business analytics and financial reporting. Built with modern web technologies for real-time insights.

## Overview

Insite Dashboard provides actionable financial metrics, KPI tracking, and business intelligence visualizations designed for CFOs and financial teams.

## Features

- **Real-time Financial Metrics** - Live KPI tracking
- **Customizable Dashboard** - Personalize your views
- **Multi-level Reporting** - Detailed financial analysis
- **Responsive Interface** - Works on desktop and mobile

## Tech Stack

- React - UI framework
- Tailwind CSS - Styling
- Recharts - Data visualization
- Vite - Build tool

## Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation

```bash
git clone https://github.com/gerrit-design/insite-dashboard.git
cd insite-dashboard
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```
